# Stock Portfolio Tracker
A lightweight, terminal-based Python utility designed to help users track their stock investments. 
This tool allows you to input your holdings based on a set of pre-defined market prices, 
calculates your total portfolio value, and exports a professional report in multiple formats

# Features
Real-time Value Calculation: Automatically calculates total investment per stock and overall portfolio value.
Data Validation: Includes robust error handling for invalid stock symbols, non-numeric inputs, and negative share quantities.
Multi-Format Export:
Text (.txt): A clean, human-readable summary with timestamps.
CSV (.csv): Standardized format ready for import into Excel, Google Sheets, or Apple Numbers.
User-Friendly Interface: Clean ASCII-art headers and clear instructional prompts.

# Installation & Usage
Prerequisites
Python 3.x installed on your machine.

Running the Program
Download the stock_tracker.py file.

Open your terminal or command prompt.

Navigate to the folder containing the file.

Run the following command:
python stock_tracker.py
