 # Hangman: Python Edition

A classic word-guessing game built using Python. 
This project demonstrates fundamental programming concepts such as 
loops, list manipulation, conditional logic, and randomization.

##  How to Play

The objective is simple: guess the secret word one letter at a time before you run out of lives.

1. The computer picks a random word from a predefined list (e.g., python, variable, syntax).
2. You are shown a series of underscores _ _ _ _ representing the length of the word.
3. Guess a letter:
 Correct:The letter fills in the appropriate blanks.
 Incorrect:* You lose one life, and a part of the "hangman" is drawn.


4. Win: Guess all the letters in the word.
5. Lose:*The hangman drawing is completed after 6 wrong guesses.

## Getting Started

### Prerequisites

 Python 3.x installed on your system.

### Running the Game

1. Copy the code into a file named hangman.py.
2. Open your terminal or command prompt.
3. Run the script using:
# bash
python hangman.py
